# Heaven-Dream
## Heaven Dream: Split World (ver 1.16.*)

在魔幻和现实中交织着，作为一场梦幻般的旅途，天堂之梦向我们打开了大门。
随着1919年的大战结束，神圣罗马帝国的对外团结和战争潜力超越了豺狼们的想象，最终在三年的血战中促成了东线战场的终结并在之后将法国重创。不列颠帝国停止了无休的封锁并退出战争，并重新构成了世界的拼图。
战间期的革命，再殖民并未彻底引爆矛盾，相反――随着萧条的过去，各个国家积蓄已久的矛盾将以更猛烈的形式爆发，而不甘心被淹没的人则与之相抗。
英法帝国的秩序起起伏伏，西地中海的半岛歌剧缓缓上演，巴尔干的火药硝烟以民族的名义引爆，北海正逐步走向迟到的尽头，而新生于夹缝中的国家则磨刀霍霍。
俄罗斯的二次革命逐步扩大，中东的各路人马趁乱争霸，印度的南北僵局摇摇欲坠，而随着旧日的中华帝国崩塌，留下的争端在整个东方的势力圈内无休止放大。
中美的沉睡国度正逐步苏醒，南美帝国的和平到了最后的时间，而北美的人民们正尝试开创属于他们自己的未来――当然还有愈演愈烈的非洲大地，埋藏着属于大陆的故事。
